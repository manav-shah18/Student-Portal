# Student Portal Web Application

A full-stack Student Portal web application designed to manage academic activities such as attendance tracking, dashboards, performance analytics, and authentication.

---

## Features

* User Authentication (Login System)
* Student Dashboard
* Teacher Dashboard
* Performance Graphs & Analytics
* Attendance Management
* Percentage Calculator
* Responsive UI using HTML, CSS, and JavaScript

---

## Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Node.js

**Other Tools:**

* Express.js

## Installation & Setup

1. Clone the repository

```
git clone https://github.com/manav-shah18/Student-Portal
```

2. Navigate to the project folder

```
cd student-portal
```

3. Install dependencies

```
npm install
```

4. Create a `.env` file in the root directory and add required environment variables

5. Run the server

```
node server.js
```

6. Open in browser:

```
http://localhost:3000
```
